# 便利店小店主 - MVP

## 游戏说明

简单的模拟经营游戏，玩家经营一家便利店：

- **商品管理**：调整商品价格、进货
- **顾客系统**：顾客会自动购买商品
- **经济系统**：赚取金币，升级设施
- **设施升级**：升级收银台和货架

## 运行方式

### 方式1：直接在浏览器打开
1. 下载 `便利店小店主.html` 文件
2. 双击打开或在浏览器中打开

### 方式2：使用本地服务器
```bash
# 使用 Python
python -m http.server 8000

# 使用 Node.js
npx serve

# 使用 PHP
php -S localhost:8000
```
然后访问 http://localhost:8000

## 部署方式

### GitHub Pages
1. Fork 或上传到 GitHub 仓库
2. 启用 GitHub Pages
3. 访问 https://your-username.github.io/your-repo/

### Vercel
1. 安装 Vercel CLI: `npm i -g vercel`
2. 部署: `vercel`
3. 访问提供的链接

### Netlify
1. 上传到 Netlify Drop
2. 访问 https://app.netlify.com/drop
